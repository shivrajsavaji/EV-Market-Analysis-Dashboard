# EV-Market-Analysis-Dashboard

# Overview

Developed an interactive Power BI dashboard to analyze Electric Vehicle (EV) market trends and sales performance using Power BI.

# Tools & Technologies

 - Power BI
 - Power Query
 - DAX
 - Microsoft Excel
  
# Key Features
 - Interactive KPI dashboard
 - Brand-wise sales analysis
 - Market trend visualization
 - Dynamic filters and slicers
 - Data cleaning using Power Query
